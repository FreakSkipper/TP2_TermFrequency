Utilização:
	* Instale o NodeJs
		- https://nodejs.org/en/download/
		- Linux : apt-get install npm
	* No terminal, digite:
		- npm install
		- npm start

Código:
	* Todo o código de KickForward está em src/kickforward.js
	* HTML/CSS serve somente para a interface gráfica.
	* Todo o processo do KickForward inicia-se na função main()
