	* O codigo foi desenvolvido com vase no paradigma Introspective designado
	pelo professor em sala de aula. Para realizar a aplicacao deste no 
	algoritmo de TermFrequency foi escolhida a linguagem Java. Por meio da
	API Reflection e acessando a StackTrace o grupo conseguiu realizar uma
	simples aplicacao do Introspection no codigo.
	
	* Introspection eh a habilidade de uma funcao obter informacoes de si mesma
	em tempo de execucao.
	
	* Reflection eh a capacidade de acessar e modificar objetos em tempo de
	execucao. Alem disso, em Java, essa API permite invocar metodos de um
	objeto, mudar acessibilidade, etc.
